== Description ==

Using categories as tags in WordPress. This plugin provides a tag cloud and related posts. When using this plugin, you can use the WordPress’ categories as tags and no additional tagging plugin is needed.

Please visit [the official website](http://sw-guide.de/wordpress/plugins/category-tagging/ "Category Tagging") for further details and the latest information on this plugin.
